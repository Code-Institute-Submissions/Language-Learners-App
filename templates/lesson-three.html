Lesson 3: Functions

Functions & Parameters

A function in JavaScript is a procedure of statements that performs a task or calculates of a value

function theFunction(a, b) {
return a * b;
}
console.log(theFunction(2, 5));

Result will be 10.

In the example above, a and b are the parameters that are returned to multiply two values. Then, the call of the function, "theFunction(2, 5);", specifies what numbers a and b should represent in this example.

Return Statement

The return statement causes the execution of a function to stop and returns a value from it.

If we use the example above:

function theFunction(a, b) {
return a * b;
}
console.log(theFunction(2, 5));

The return statement stops the function and it returns and specifies what the parameters of the function should do.
