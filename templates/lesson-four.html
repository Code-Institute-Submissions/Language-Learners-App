Lesson 4: Objects

Methods

// For assistance of examples and explanations below, I used this link: https://www.w3schools.com/js/js_object_methods.asp

Methods are used for objects and to do something with the information inside of them. Here is are some examples

let pet = {
animal: "cat",
name: "catty",
age: 5,
// ObjectName.Method is the format console.log(pet.animal)

The code above results in, "cat"

One could also turn all letters of it to capital letters, like so:

let animalType = pet.animal
console.log(animalType.toUpperCase())

The result is: "CAT"

There are many other methods you can look up.
