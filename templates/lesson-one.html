Lesson 1: Basics

Variables, Constants and Let

One can define a values through the following three terms:

  • var
  • let
  • const

For example, one could write:

var x = 10;

var y = 5;

var z = x + y

In this example, z is equal to 15, as it is the result of 10 plus 5.

In order to display the z variable in the console of Google Chrome, use console.log, as in console.log(z) and 15 should be shown in the console.

Nowadays, let is used more often instead of var and const is only used, if one wants a value to stay the same.

Also, please note that a semicolon (;) is usually included at the end of each line.

Data Types

In JavaScript, there are many multiple data types, such as strings, array, objects, numbers and booleans. Here are some examples:

  • let theString = "This is a string";
  • let theArray = ["This", "is", "a", "list", "of", "words"];
  • let theObject = {device: "Mobile Phone", companyName = "Samsung", version = "Galaxy S8"};
  • let theNumber = 63 / 3;
  • let theTrueBoolean = True;
  • let theFalseBoolean = False;

As it is shown above, a string is a text in quotation marks, whereas an array is a list of things, which can be letters, numbers or symbols.

The object has pairs that have a name and a value and they are all wrapped in curly brackets ({}). The values could be targetted as in the following example:

console.log("The " + theObject.device + " is a " + theObject.companyName + " " + theObject.version;)

A boolean can either be true or false. For instance, if the following two variables were compared:

  • let theNumber = 63 / 3;
  • let theOtherNumber = 24 - 3;
  • (theNumber == theOtherNumber)

In the example above, (theNumber == theOtherNumber) would return true because both variables are equal to 21, but if they would result in different numbers, it would return false.
