Lesson 6: Using the DOM

Selection

You can get IDs from HTML, like so:

let theId = document.getElementById("HTMLIdName")

A HTML class can be grabed through this:

document.getElementsByClassName("HTMLClassName")

There are many more examples that you can look up.

Event Listeners

Event listeners trigger a function, through an action, like a click or hover. Here is the main structure of it:

let variable = document.getElementById("elementId")
variable.addEventListener("click", functionName());

In this example, "click" indicates that the event the functionName() function shall be triggered by is through a click.
