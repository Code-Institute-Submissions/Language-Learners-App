Lesson 2: Loops and Contionals

If and Else Statements

if(condition) {
// code gets executed, if true
}
else if(condition) {
// code gets executed, if first condition is not true
}
else {
// code gets executed, if all else is false
}

Example:

let minutes = 30;
let hour = 1;
if (minutes == 2 && hour != 1) {
return "Two minutes have passed."
}
else if(minutes == 26 || hours == 1) {
return "More than 25 minutes have passed."
}
else {
return "Less than two minutes have passed"
}
The result will be "More than 25 minutes have passed.", as hour = 1.

Switch Statements

switch(expression) {
case x:
// code gets executed in case x matches
break; // required to stop the execution
case y:
// code is here
continue; // Means that the case will be skipped, if matched
case z:
// code gets executed, if true
break;
default:
// code that gets executed in case nothing of the above matches
}

For Loops

for (start; finish; increment/decrement) {
// Code comes here
}

let arr = [3,4,5,6,7]

for(i = 0; i < arr.length; i++) {
// Code goes here
}

forEach

If there is an array, forEach has the ability to loop through each of the items in the array!

Here is an example:

let countries = ["Ireland", "Indonesia", "Dubai"];

function TheFunction(country) {

console.log(country);

}

countries.forEach(TheFunction)

This logs:

Ireland

Indonesia

Dubai

While Loops

While Loop

The while loop works like a for loop, but the while loop runs, as long as a statement is true, whereas the for loop runs a number of iterations.

let i = 0;

while (i < 3) {}

console.log(i++);

}

The result logs:

0

1

2

Do/While Loop

There is also a do/while loop which executes code before checking if the condition in the while loop is true

It is formated like this:

let i = 0;

do {

// Code comes here

} while(i < 3);
