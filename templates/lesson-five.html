Lesson 5: Arrays

Arrays

Arrays in JavaScript are like lists that consist of multiple value in them.

So, instead of having multiple variables, like this:

let participant1 = "Justin"
let participant2 = "Mina"
let participant3 = "Alex"

We could add them into one array:

let participants = ["Justin", "Mina", "Alex"]
console.log(participants[0])

The result of this is: "Justin"
